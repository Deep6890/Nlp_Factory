# 🏆 Lazy Legends - Multi-Platform Project

> **Hackathon Project**: Combined Flutter Mobile App + React Web Frontend

## 📱 Project Structure

```
lazy-legends/
├── my_app/           # 📱 Flutter Mobile Application
│   ├── lib/          # Dart source code
│   ├── android/      # Android build files
│   ├── ios/          # iOS build files
│   └── server/       # Node.js backend
│
├── armor-project/    # 🌐 React Web Application
│   ├── src/          # React source code
│   ├── public/       # Static assets
│   └── build/        # Production build
│
└── public/           # Shared assets
```

## 🚀 Quick Start

### Flutter App (my_app)
```bash
cd my_app
flutter pub get
flutter run
```

### React App (armor-project)
```bash
cd armor-project
npm install
npm start
```

## 🛠️ Tech Stack

- **Mobile**: Flutter + Dart
- **Web**: React + JavaScript
- **Backend**: Node.js
- **Styling**: Tailwind CSS
